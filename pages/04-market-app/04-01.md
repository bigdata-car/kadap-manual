

# 마켓 플레이스 KADaP MARKET



자동차 데이터 기반의 분석 및 개발에 필요한 다양한 데이터뿐만 아니라, APP과 API 형태의 상품이나 서비스를 간편하게 구매하고 활용할 수 있습니다.



# 마켓 플레이스 소개



마켓 플레이스는 KADaP 내에서 운영되는 데이터 기반 거래 공간으로, 기업과 기관이 데이터를 공유하거나 판매하고, 필요한 데이터를 구매하는 등 데이터 상품의 거래와 서비스 환경을 제공합니다.



마켓 플레이스는 SaaS 마켓 플레이스와 API 마켓 플레이스로 구성되어 있고, 각 마켓 플레이스의 서비스 구성과 주요 특징은 다음과 같습니다.

## 주요 기능과 특징



### SaaS 마켓 플레이스



- **APP 검색 및 탐색 기능 제공**



&#x20; 다양한 자동차 기술 분야의 APP을 카테고리별로 분류하여 쉽게 찾아볼 수 있습니다.



- **APP 이용 신청**



&#x20; 원하는 APP을 선택하여 이용 신청하고, 승인 후 바로 사용할 수 있습니다. 무료 APP은 이용 신청을 하면 바로 사용 가능하고, 유료 APP은 개발자 승인 후 사용할 수 있습니다.



- **포인트 기반 과금**



&#x20; 포인트(무료 APP)·포인트 플러스(유료 APP)를 통해 사용 시간에 따라 차감됩니다.



- **APP 등록 및 게시 기능 제공**



&#x20; 개발자는 자체 개발한 APP을 등록하고, 관리자 승인 후 마켓에 공개할 수 있습니다.



- **승인 기반 운영 체계**



&#x20; 관리자의 검토 및 승인 절차를 통해 신뢰할 수 있는 APP만 마켓에 공개됩니다.



- **대시보드**



&#x20; 사용 중인 APP 목록과 이용 상태를 한눈에 확인할 수 있습니다.
