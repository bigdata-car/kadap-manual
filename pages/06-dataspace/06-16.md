\### 프로젝트 데이터 관리하기 {#프로젝트-데이터-관리하기}



자동차데이터플랫폼(KADaP) 클라우드에 저장된 장치별 데이터를 확인하고 관리할 수 있습니다.



프로젝트 데이터를 관리하려면 다음 순서대로 진행하세요.



1\. 데이터 수집 시스템의 홈 화면에서 \*\*프로젝트 목록\*\*에서 원하는 프로젝트를 클릭하세요.

2\. 프로젝트 관리 메뉴에서 \*\*프로젝트 데이터 관리\*\*를 클릭하세요.

3\. 프로젝트 데이터 설정 화면에서 root 경로에 저장된 데이터를 확인하세요.



&#x20;  !\[3486.png](./../../assets/06-dataspace/3486.png)



&#x20; - \*\*파일형식\*\*을 클릭해 필터링 항목을 선택하고 \*\*Apply\*\*를 클릭하면 해당 포맷 파일만 확인할 수 있습니다.

&#x20;   - 검색창 하단에 선택한 파일형식 항목이 표시됩니다.

&#x20;   - 파일형식 항목의 \*\*X\*\*를 클릭하거나 \*\*초기화\*\*를 클릭하면 선택한 항목을 해제할 수 있습니다.

&#x20; - 폴더 목록 중 \*\*output\*\*폴더 > 장치별 폴더에 전처리 모델로 처리된 데이터가 저장됩니다.

&#x20; - \*\*폴더/파일 목록\*\*의!\[icon29\_fmt.PNG](./../../assets/06-dataspace/icon29\_fmt.png)를 클릭하면 항목을 추가할 수 있습니다.

&#x20;   - 폴더 생성 창이 열리면 폴더명을 입력하고 \*\*생성\*\*을 클릭해 추가합니다.

&#x20;   - 파일 업로드 창이 열리면 업로드할 파일을 선택하고 \*\*업로드\*\*를 클릭해 추가합니다.

&#x20; - \*\*폴더/파일 목록\*\*의 !\[icon24\_fmt.PNG](./../../assets/06-dataspace/icon24\_fmt.png)를 클릭하면 선택한 항목을 삭제할 수 있습니다.

&#x20;   - 파일 항목의 \*\*다운로드\*\*를 클릭하면 해당 파일을 PC로 다운로드할 수 있습니다.
