
## 수집 단말기 세팅 하기 

차량 및 설비 데이터 수집 단말기가 데이터 수집 시스템에 연결되기 위해서는 제공되는 소프트웨어 설치가 필요 합니다. 

[[TIP("설치 요구 사항)]]
- 리눅스 : ubuntu 계열 추천
- K3s 설치가 가능한 환경
[[/TIP]]

설치를 위한 스크립트를 제공하여 쉽게 진행 할 수 있습니다. 

### 설치 하기 

#### 1. 설치 스크립트 다운로드 

1. 설치 스크립트 다운로드 [[다운로드][(./../../assets/06-dataspace/install_iot.zip)
2. 단말기로 복사 
3. 압축 풀기 `unzip ./install_iot.zip`

#### 2. 설정 정보 입력 하기 

```
vi run.sh
	"""
  --tls-san 192.168.0.115 :  외부에서 장비에 접근하기 위한 IP 설정 (공인 IP 추천)
  --generate-kubeconfig : kubeconfig 파일 생성 설정, 장치 등록시 필요 
  --kubeconfig-out ~/edge5.kubeconfig.yaml : 생성 할 kubeconfig 파일명 
  --install-n8n : 노코딩 툴(n8n) 설치시 지정  
  --n8n-host n8n.default.local : 노코딩 툴(n8n) 호스트 설정, 기본값 그대로 사용, 
  --n8n-iframe-allow "https://kadap-edge.quantum-x.co.kr" : iframe 사용지 접근 주소 설정 
	"""
```

[[TIP("중요)]]
공인 IP가 아닌 사설IP를 사용 할 경우에는 포트포워딩으로 외부 접속을 허용 해야 합니다. 
- 22번 포트 : 단말기내 폴더 정보 열람용(SFTP)
- 6443번 포트 : 자동차데이터플랫폼과 통신용
[[/TIP]]

#### 3. 설치 스크립트 실행 

```bash
chmod +x insatall-k3s.sh
chmod +x run.sh
sudo ./run.sh
```
