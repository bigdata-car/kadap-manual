## 장치 관리하기

[TOC]



데이터를 수집할 장치를 등록하고 관리할 수 있습니다.

### 화면 구성

![[Pasted image 20260703211717.png]]

데이터 수집 시스템의 **장치 관리** 메뉴에서 장치를 등록하고 관리할 수 있습니다.



![3356.png](./../../assets/06-dataspace/3356.png)



- 장치 관리 항목에서는 장치를 수정하거나 삭제할 수 있습니다.

- **수정**: 장치 추가 화면으로 이동해 장치 정보를 수정할 수 있습니다.

- **삭제**: 등록한 장치를 삭제합니다. 삭제한 장치는 복구할 수 없습니다.



### 장치 추가하기


1. 데이터 수집 시스템의 홈 화면에서 **장치 관리**를 클릭하세요.

2. 장치 관리 화면에서 **+ 생성**을 클릭하세요.

3. 장치 추가 화면에서 상세 정보를 입력하고 **생성**을 클릭하세요.

![[Pasted image 20260703210048.png]]



데이터 수집 장치를 등록하려면 다음 순서대로 진행하세요.

| 입력창                                  | 설명                                                                                                                                                                                                                                                                                      |
| ------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| ![[Pasted image 20260703210437.png]] | - **기본 정보**<br><br>- **장치명**: 추가할 장치 이름을 입력합니다.<br><br>- **설명**: 장치 설명을 입력합니다.<br><br>- **IP**: 장치의 IP 주소를 입력합니다.                                                                                                                                                                       |
| ![[Pasted image 20260703210504.png]] | - **장치유형 정보**<br><br>- 전처리: 수집된 데이터에 전처리 모델을 적용해 표준 언어로 변환합니다.<br><br>- CAN: 차량 제어 유닛(ECU)에서 발생하는 다양한 정보를 실시간으로 수집합니다.<br><br>- 센서: 제조 현장의 설비 상태나 특정 부품 상태를 측정해 실시간 데이터를 수집합니다.                                                                                                         |
| ![[Pasted image 20260703210522.png]] |                                                                                                                                                                                                                                                                                         |
| ![[Pasted image 20260703210555.png]] | - **SFTP 접속 정보**<br><br>장치의 SFTP(Secure File Transfer Protocol) 접속 정보를 입력합니다. 입력한 SFTP로 수집된 데이터가 암호화되어 전송됩니다.<br><br>- **SFTP PORT**: SFTP 접속 포트를 입력합니다. (기본 포트: 22)<br><br>- **SFTP ID**: SFTP 서버에 접속할 때 필요한 ID 정보를 입력합니다.<br><br>- **SFTP PASSWORD**: SFTP 서버에 접속할 때 필요한 비밀번호를 입력합니다. |
| ![[Pasted image 20260703210626.png]] | - **K8S 접속 정보**<br><br>- **K8S PORT**: 장치의 K8S 접속 포트를 입력합니다. (기본 포트: 6443)<br><br>- 장치의 K8S 클러스터에 접속하기 위한 **kubeconfig** 파일을 업로드합니다. 이 파일에는 API 서버 주소와 인증 정보가 포함되어 있습니다.                                                                                                                |
| ![[Pasted image 20260703210631.png]] | - **동기화 정보**<br><br>- **동기화 주기(초)**: 장치와 연결된 클라우드 서버나 저장소와의 동기화 주기를 입력합니다. (기본 동기화 주기: 3600)                                                                                                                                                                                            |
| ![[Pasted image 20260703210642.png]] | - **위치 정보**<br><br> - 검색창에 도로명 주소를 입력하고 **검색**을 클릭하면 지도 위에 위치창이 나타납니다. 위치창의 주소 정보를 확인하고 **선택**을 눌러 장치 위치를 등록합니다.                                                                                                                                                                        |
|                                      |                                                                                                                                                                                                                                                                                         |
