# 데이터스페이스 포털



일반 사용자는 마켓 플레이스에서 앱을 구독할 수 있으며, 구독 신청한 앱 정보와 자동차데이터플랫폼(KADaP)에 등록된 기업 정보를 조회할 수 있습니다. 서비스 제공자인 경우 마켓 플레이스에 앱 등록 및 데이터 서비스를 등록하고 관리할 수 있습니다.



> ![4736.png](./../../assets/06-dataspace/4736.png) **참고**

>

> 데이터 교환 시스템 포털에는 기관 회원 중 기업 정보 등록이 완료된 사용자만 접속할 수 있습니다.

> - 포털에 접속이 가능한 사용자만 데이터 수집 시스템(IoT)이나 데이터 교환 시스템(Edge)을 사용할 수 있습니다.



## 홈 화면 구성



데이터 교환 시스템 포털의 홈 화면은 다음과 같이 구성됩니다. 각 항목별 기능을 설명합니다.



![2570.png](./../../assets/06-dataspace/2570.png)



| 번호 | 항목 | 설명 |

| --- | --- | --- |

| 1 | 데이터 교환 시스템 포털 메인 메뉴 | 데이터 교환 시스템 포털의 메뉴를 선택합니다.<ul><li>**홈**: 데이터 교환 시스템의 홈 화면으로 이동합니다.</li><li>**마켓 플레이스**: 포털에 등록된 앱의 상세 정보를 확인하고 구독 신청을 할 수 있습니다.</li><li>**데이터 관리**: 데이터 제공자에게 시맨틱 모델을 제공하고, 필요한 경우 등록 신청할 수 있습니다.</li><li>**파트너 네트워크**: 포털에 등록된 회사 정보를 조회할 수 있습니다.</li><li>**앱 관리**: 회사 관리자인 경우 앱을 등록하고 관리할 수 있습니다.</li></ul>|

| 2 | 통합 검색 | 데이터 교환 시스템 포털의 정보를 검색할 수 있습니다. |

| 3 | 즐겨찾기 | 사용자가 구독하는 앱 중 즐겨찾기가 설정된 앱 목록이 표시됩니다. |

| 4 | 내 정보 | 로그인한 사용자 정보를 확인하고 알림을 확인할 수 있습니다.<ul><li>내 정보의 상세 메뉴는 사용자 권한에 따라 달라집니다. 관리자가 사용할 수 있는 메뉴에 대한 자세한 설명은 [내 정보 설정하기](#내-정보-설정하기)를 참고하세요.</li><li>**나의 계정**: 포털에 등록된 내 계정 정보를 확인할 수 있습니다.<ul><li>**TOKEN을 클립보드로 복사**: 사용자 계정의 토큰 데이터를 복사합니다. 사용자 토큰은 사용자 인증 시 적용할 수 있습니다.</li><li>**사용자 삭제**: 내 정보를 삭제합니다.사용자 삭제는 관리자 승인 후 적용됩니다.</li></ul><li>**나의 조직**: 사용자가 소속된 회사 정보 및 앱 구독 목록을 확인할 수 있습니다.</li><li>**알림**: 포털 공지사항이나 사용자 알림을 확인할 수 있습니다.</li><li>**로그아웃**: 사용자 계정에서 로그아웃합니다.</li></ul>|

| 5 | 언어 설정 | 데이터 교환 시스템 포털의 표시 언어를 국문/영문 중에서 설정할 수 있습니다. |

| 6 | 바로가기 | 데이터 교환 시스템 포털에서 가장 많이 사용하는 메뉴의 바로가기를 제공합니다.<ul><li>**나의 앱**: 로그인한 사용자의 조직 정보와 앱 구독 목록을 확인할 수 있는 나의 조직 화면으로 이동합니다.</li><li>**마켓 플레이스**: 앱 마켓 플레이스 화면으로 이동합니다.</li><li>**포털 사용자 관리**: 회사 관리자인 경우 포털 사용자 관리 화면으로 이동합니다.</li></ul>|

| 7 | 데이터 교환 시스템 포털 최신 정보 | 데이터 교환 시스템 포털의 최신 업데이트, 릴리즈 정보를 확인할 수 있습니다. |

| 8 | 구독 앱 | 사용자가 구독 중인 앱 목록을 확인할 수 있습니다. <ul><li>앱이 연동된 경우 해당 앱을 클릭하면 서비스 홈 화면으로 바로 이동합니다.</li></ul>|

| 9 | 신규 출시 앱 | 데이터 교환 시스템 포털에 신규 등록된 앱 목록을 확인할 수 있습니다. |
