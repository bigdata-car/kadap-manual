\### 파일 업로드 및 파싱하기



마이 챗봇을 생성하기 위해서는 파일을 업로드하고 해당 파일을 AI가 인식할 수 있는 형태로 변경해야 합니다. 파싱은 이미지나 PDF 파일 내의 텍스트를 AI가 읽을 수 있도록 변환합니다.



파일을 업로드하고 파싱하려면 다음 순서대로 진행하세요.



1\. \*\*자료 관리\*\* 페이지에서 \*\*내 자료\*\* 탭을 클릭하세요.

2\. \*\*파일 업로드\*\*를 클릭하세요.

&#x20;  - \*\*파일 업로드 및 파싱\*\* 팝업창이 표시됩니다.



&#x20;  !\[1474.png](./../../assets/07-agent/1474.png)



3\. \*\*파일 선택하기\*\*를 클릭하여 원하는 파일을 선택한 후, \*\*파일 업로드\*\*를 클릭하세요.



&#x20;  !\[1492.png](./../../assets/07-agent/1492.png)



&#x20;  - 10 MB 이하의 파일만 업로드할 수 있습니다.

4\. 문서 특성에 맞는 원하는 파서를 선택한 후, \*\*파싱 시작\*\*을 클릭하세요.

&#x20;  - 문서 파싱을 진행합니다.<br>파일 크기나 서비스 제공자의 상황에 따라 시간이 소요될 수 있습니다.



&#x20;   !\[1525.png](./../../assets/07-agent/1525.png)



&#x20;  - 유료 파서의 경우, 기본 제공 횟수를 소진하면 다음의 경로에서 사용자의 파서 API 키를 등록하여 계속해서 사용할 수 있습니다.사용자 프로필 클릭 > \*\*대시보드\*\* > \*\*파서 API 키\*\* > \*\*새 API 키 추가\*\*

&#x20;  - \*\*건너뛰기\*\*를 클릭하면 문서 파싱을 하지 않고 파일 업로드만 진행합니다. 업로드한 파일은 자료 목록에서 확인할 수 있고, 필요한 경우 !\[Icon\_Agent\_3.png](./../../assets/07-agent/Icon\_Agent\_3\_fmt1.png)을 클릭하여파싱을 진행할 수 있습니다.

5\. 문서 파싱이 완료되면 자료 목록에 폴더와 추출된 상세 파일이 생성됩니다.

&#x20;  - 파싱된 파일은 md, html, json, png 형식으로 저장됩니다.

&#x20;  - 파싱 결과를 지원하는 일부 파서의 경우 \*\*파싱 결과 시각화\*\* 창이 추가로 표시됩니다.

&#x20;  - 파싱된 파일은 다운로드하여 다른 LLM에서 활용할 수 있습니다.



\### 파싱 결과 시각화 화면 확인하기



파싱을 진행한 문서가 PDF 파일이면 시각화 창을 통해 파싱 결과를 확인할 수 있습니다.



!\[1548.png](./../../assets/07-agent/1548.png)



\- 하이라이트되지 않는 엘리먼트는 인식되지 않은 부분으로 AI 응답에 활용될 수 없습니다.

> !\[2611.png](KADaP\_UserManual\_Frontmatter-web-resources/image/2611.png) \*\*참고\*\*

>

> 파싱 결과 시각화는 일부 파서에서만 지원하고 있습니다. (예: Updatage)
