\## 대시보드



자동차 지식 에이전트에서는 대시보드를 통해 사용자 활동 내역과 리소스 사용량을 모니터링할 수 있습니다. 또한 문서의 파싱 및 분석을 위한 카테고리별(LLM 또는 파서) API 키를 추가하거나 변경하여 관리할 수 있고, 데이터 특성에 따라 파서를 변경하여 분석 내용의 품질을 향상시킬 수 있습니다.



자동차 지식 에이전트의 로그인 계정을 클릭한 후, \*\*대시보드\*\*를 클릭하세요. 대시보드 페이지로 이동합니다.



!\[1711.png](./../../assets/07-agent/1711.png)



\### 화면 구성



대시보드 화면은 다음과 같이 구성됩니다.



!\[1723.png](./../../assets/07-agent/1723.png)



| 번호 | 항목 | 설명 |

| --- | --- | --- |

| 1 | 내 보고서 | 사용자가 작성한 보고서 개수를 확인할 수 있고, 위젯을 클릭하면 \*\*보고서\*\* 페이지로 이동합니다.<ul><li>\*\*최근 보고서\*\* 목록에서 원하는 보고서를 클릭하면 해당 보고서로 이동합니다.</li></ul> |

| 2 | 내 채팅 | 챗봇과의 대화 개수를 확인할 수 있고, 위젯을 클릭하면 \*\*챗봇\*\* 페이지로 이동합니다.<ul><li>\*\*최근 채팅\*\* 목록에서 원하는 보고서를 클릭하면 해당 보고서로 이동합니다</li></ul> |

| 3 | 내 자료 | 사용자가 업로드한 자료의 개수를 확인할 수 있고, 위젯을 클릭하면 \*\*자료 관리\*\* 페이지로 이동합니다. |
