\# 서버 생성 및 관리 하기 



\## 서버 관리하기



서버 관리 페이지에서 사용자가 생성한 전체 서버 목록을 확인하고 관리할 수 있습니다.



\### 카드 정보 및 메뉴 설명 {#카드-정보-및-메뉴-설명}



서버 종류별로 카드에 표시되는 정보와 상세 메뉴를 설명합니다.



> !\[2043.png](./../../assets/02-cloud/2043.png) \*\*참고\*\*

>

> 선택한 서버의 OS 및 사양에 따라 사용할 수 있는 메뉴와 표시 항목이 달라집니다.



!\[2968.png](./../../assets/02-cloud/2968.png)



\- \*\*OS 라이선스 표시\*\*

&#x20; - \*\*Free\*\*: 무료 사용 라이센스

&#x20; - \*\*Paid\*\*: 구매 후 제공되는 라이센스

&#x20; - \*\*BYOL\*\*: 사용자 보유 라이선스를 입력하여 사용



\- \*\*서버 제어 메뉴\*\*

&#x20; !\[Icon\_Cloud\_13.png](./../../assets/02-cloud/Icon\_Cloud\_13\_fmt.png)를 클릭하면 선택한 서버의 제어 메뉴가 표시됩니다. (\[서버 제어](#서버-제어) 참고)



\- \*\*접속 정보\*\*

&#x20; 서버 접속 정보가 표시됩니다. (\[접속 정보 확인](#접속-정보-확인) 참고)



\- \*\*웹터미널\*\*

&#x20; 웹 기반 SSL 클라이언트로 서버에 접속합니다.(\[웹 터미널 접속](#웹-터미널-접속) 참고)



\- \*\*원격 데스크탑\*\*

&#x20; 원격 데스크탑에 접속합니다. (\[원격 데스크탑 연결](#원격-데스크탑-연결) 참고)



\- \*\*서버 자원 사용률\*\*

&#x20; 프로젝트의 전체 쿼터에서 서버 자원 사용률을 표시합니다.



\- \*\*서버 상태\*\*

&#x20; 현재 서버 상태를 표시합니다.

&#x20; - \*\*활성\*\*: 서버가 정상 작동중입니다.

&#x20; - \*\*비활성화\*\*: 서버가 정지되고 자원이 회수된 상태입니다.

&#x20; - \*\*정지\*\*: 서버 작동이 정지된 상태입니다.

&#x20; - \*\*미지원\*\*: 모니터링 에이전트가 설치되지 않은 경우 표시됩니다.



\### 서버 제어 {#서버-제어}



카드 정보에서 !\[Icon\_Cloud\_13\_fmt.png](./../../assets/02-cloud/Icon\_Cloud\_13\_fmt.png)를 클릭하면 서버 제어 메뉴를 사용할 수 있습니다.



!\[3016.png](./../../assets/02-cloud/3016.png)



\- \*\*서버 제어 메뉴\*\*

&#x20; - \*\*이름 변경\*\*: 서버 이름을 변경합니다.

&#x20; - \*\*시작\*\*: 정지된 서버를 부팅합니다.

&#x20; - \*\*재시작\*\*: 정지된 서버를 다시 작동시킵니다.

&#x20; - \*\*정지\*\*: 서버 작동을 정지합니다.

&#x20; - \*\*정지 \& 자원회수\*\*: 서버 작동을 정지하고 서버에 할당된 자원을 회수해 비활성화합니다.

&#x20; - \*\*정지 \& 자원회수 해제\*\*: 서버 작동을 시작하고 서버에 할당된 자원을 다시 사용합니다.

&#x20; - \*\*포트포워딩\*\*: 서버의 포트 포워딩을 설정합니다.

&#x20; - \*\*콘솔 접속 (VNC)\*\*: 가상 서버의 부팅 및 에러 등으로 접속이 제한될 경우 상태 확인을 위한 콘솔에 접속할 수 있습니다.

&#x20; - \*\*웹 콘솔 접속 (VPN)\*\*: 가상 서버의 관리를 위한 웹 콘솔 서비스에 접속할 수 있습니다. (VPN 연결 필요)

&#x20; - \*\*서버 사양 변경\*\*: 서버 자원 사양을 변경합니다. 서버가 정지된 상태에서만 사양을 변경할 수 있습니다.

&#x20; - \*\*백업 이미지 생성\*\*: 가상 서버의 현재 상태를 이미지 형태로 저장합니다.<br>저장된 이미지를 이용하여 현재 상태의 가상 서버를 생성할 수 있습니다. 서버가 정지된 상태에서만 백업 이미지를 생성할 수 있습니다.<br>자세한 설명은 \[백업 이미지로 서버 생성하기](#백업-이미지로-서버-생성하기)를 참고하세요.

&#x20; - \*\*디스크 연결\*\*: 가상 서버에 새로운 저장 공간 및 기존 저장 공간을 연결할 수 있습니다.

&#x20; - \*\*키페어 다운로드\*\*: 가상 서버 접속 시 사용하는 보안 키를 다운로드할 수 있습니다.

&#x20; - \*\*서버 삭제\*\*: 선택한 서버를 삭제합니다.<br>삭제한 서버는 복구할 수 없습니다.<br>서버 삭제 전 백업 이미지를 생성하거나 필요한 데이터를 별도로 저장하세요.

\### 서버 정보



서버의 상세 정보를 확인할 수 있습니다.



!\[3049.png](./../../assets/02-cloud/3049.png)



\- \*\*서버 정보\*\*

&#x20; 서버 정보를 표시합니다.

&#x20; - \*\*접속 IP\*\*: 서버의 접속 IP 정보를 표시합니다.

&#x20;   - VPN에 접속하면 같은 네트워크에 포함됩니다.

&#x20; - \*\*내부 IP\*\*: 서버의 내부 IP 주소로 같은 프로젝트 내 가상 서버들이 통신할 때 사용합니다.

&#x20; - \*\*키페어\*\*: !\[icon\_10\_fmt.png](./../../assets/02-cloud/icon\_10\_fmt.png)를 클릭하면 서버의 개인키를 .pem 파일로 다운로드할 수 있습니다.

&#x20; - \*\*생성일시\*\*: 서버의 생성일시 정보를 표시합니다.

&#x20; - \*\*상태\*\*: 서버의 상태를 표시합니다. 서버가 작동 중일 경우 \*\*활성\*\*으로 표시되고, 정지 중일 경우 \*\*정지\*\*로 표시됩니다.

&#x20; - \*\*모니터링\*\*: 서버 자원의 사용 현황 모니터링 상태를 표시합니다.

&#x20; - \*\*방화벽\*\*: 방화벽에 서버 등록 여부를 표시합니다.

&#x20;   - \*\*등록 필요\*\*를 클릭하면 자동으로 서버 IP 정보가 등록되며 \*\*등록 완료\*\*로 변경됩니다.



\- \*\*서버 사용량\*\*

&#x20; 서버의 현재 컴퓨팅 자원 사용률을 표시됩니다.



\- \*\*서버 제어\*\*

&#x20; 서버의 작동을 제어합니다.

&#x20; -  !\[Icon\_Cloud\_9\_fmt.png](./../../assets/02-cloud/Icon\_Cloud\_9\_fmt.png)를 클릭하면 서버가 정지되고, !\[Icon\_Cloud\_20\_fmt.png](./../../assets/02-cloud/Icon\_Cloud\_20\_fmt.png)를 클릭하면 서버가 시작됩니다.

&#x20; -  !\[Icon\_Cloud\_10\_fmt.png](./../../assets/02-cloud/Icon\_Cloud\_10\_fmt.png)를 클릭하면 추가 메뉴가 표시됩니다. 서버 제어 기능에 대한 자세한 설명은 \[서버 제어](#서버-제어)를 참고하세요.



\### 디스크 연결 정보 {#디스크-연결-정보}



서버에 연결된 디스크 목록을 확인할 수 있습니다.



\- 디스크를 연결하려면 !\[Icon\_Cloud\_10\_fmt.png](./../../assets/02-cloud/Icon\_Cloud\_10\_fmt.png) > \*\*디스크 연결\*\*을 클릭하세요. 디스크 연결창에서 디스크를 선택해 연결할 수 있습니다.

\- !\[Icon\_Cloud\_11\_fmt.png](./../../assets/02-cloud/Icon\_Cloud\_11\_fmt.png) > \*\*연결해제\*\*를 클릭하면 서버와 연결을 해제할 수 있습니다.

\- 디스크를 신규 추가하려면 \[디스크 생성하기](#디스크-생성하기)를 참고하세요.



&#x20; !\[3151.png](./../../assets/02-cloud/3151.png)



\### 서버 상태 탭



서버의 이벤트 발생 이력, 부트 로그, 시스템 로그, 모니터링 정보를 확인할 수 있습니다.



\- \*\*서버 이벤트\*\*: 사용자가 서버에서 실행한 시작, 정지 등 액션 실행 이력을 표시합니다.



&#x20;  !\[3162.png](./../../assets/02-cloud/3162.png)



\- \*\*서버 부트 로그\*\*: 서버가 시작될 때의 부팅 로그를 표시합니다.



&#x20;  !\[3178.png](./../../assets/02-cloud/3178.png)



\- \*\*시스템 로그\*\*: 서버의 시스템 로그를 표시합니다.<br>조회 범위를 선택해 로그를 검색할 수 있습니다.



&#x20;  !\[3194.png](./../../assets/02-cloud/3194.png)



\- \*\*가상머신 모니터링\*\*: 서버의 자원별 사용률을 그래프로 표시합니다.<br>조회 범위를 선택해 사용률을 조회할 수 있습니다.



&#x20;  !\[3207.png](./../../assets/02-cloud/3207.png)



\- \*\*알람 이벤트\*\*: 사전에 설정한 알람 이벤트를 조회할 수 있습니다.

&#x20; - 알람 설정은 상단 바의 !\[Icon\_Cloud\_12\_fmt.png](./../../assets/02-cloud/Icon\_Cloud\_12\_fmt.png) > \*\*알람 더보기+\*\* > \*\*알람설정\*\* 페이지에서 할 수 있습니다.



&#x20; !\[3233.png](./../../assets/02-cloud/3233.png)
