### GPU 사용 연장하기 {#gpu-사용-연장하기}



현재 사용자가 사용 중인 GPU의 사용 시간 연장을 신청할 수 있습니다.



> ![4513.png](KADaPUserManualGPUaaS-web-resources/image/4513.png) **참고**

>

> GPU 사용 연장은 최대 3회까지만 신청할 수 있습니다. 단, GPU의 사용 예약 대기자가 있는 경우에는 사용 연장을 신청할 수 없습니다.추가 사용 연장이 필요한 경우 관리자에게 메일로 문의하세요.

> - 관리자 메일 주소: admin@bigdata-car.kr



GPU 사용 연장을 신청하려면 다음 순서대로 진행하세요.



1. 인공지능 개발 플랫폼 홈 화면에서 현재 사용 중인 GPU 슬롯을 클릭하세요.

2. 왼쪽의 GPU 사용 정보에서 **시간 연장**을 클릭하세요.

3. 워크로드 사용 시간 연장창이 나타나면 종료 예정시간을 확인하고 추가 사용시간을 선택하세요.

- 사용자가 보유한 포인트 내에서 추가 사용시간을 설정할 수 있습니다.

- 추가 사용시간은 한 시간 단위 또는 10분 단위로 추가 설정할 수 있습니다.



![2704.png](KADaPUserManualGPUaaS-web-resources/image/2704.png)



4. **사용 시간 연장**을 클릭하세요.

5. 사용 시간 연장 확인창이 나타나면 **확인**을 클릭하세요.<br> 사용 시간 연장 신청이 완료되면 왼쪽 사용 정보의 연장횟수 항목에 신청 횟수가 표시됩니다.
