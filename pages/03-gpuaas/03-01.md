# 인공지능 개발 플랫폼 KADaP GPUaaS

[TOC]

고가의 서버와 GPU 구축 없이도 AI 개발이 가능합니다.인공지능 개발솔루션은 필요한 시점에만 접속해 사용하는 방식으로 인프라 부담을 줄이고, 자동차데이터플랫폼과 연계된 최적의 개발 환경을 제공합니다.



## 인공지능 개발 플랫폼 소개



인공지능 개발 플랫폼에서는 AI 모델을 만들고 학습시킬 수 있는 통합 개발 환경 (IDE, Integrated Development Environment)을 제공합니다. GPUaaS (GPU as a Service) 기반으로 GPU 서버를 구축하지 않고 클라우드에서 필요한 만큼 GPU 리소스를 할당받아 사용할 수 있어 효율적으로 개발을 진행할 수 있습니다.



사용자는 인공지능 개발 플랫폼에서 원하는 AI 모델을 생성하고 검증하며, 데이터셋을 사용해 AI 모델을 학습시킨 후 결과를 확인할 수 있습니다. 또한 AI 기술 개발을 위한 코드, 데이터셋, GPU 학습 관리가 편리하도록 ML/AI 프레임워크와 통합개발환경을 연계해 제공합니다.



### 서비스 구성



인공지능 개발 플랫폼의 주요 서비스 구성은 다음과 같습니다.



![2448.png](./../../assets/03-gpuaas/2448.png)

워크스페이스 : 프로젝트의 개념으로 동일한 업무를 수행하는 그룹 단위 입니다. 팀 구성원을 추가 하여 소스코드, 학습데이터, GPU 자원을 공유 하고 협업이 가능합니다. 
- 사용자는 가입시 두개의 워크스페이스가 생성 됩니다.
	- 보유 GPU & 공유GPU(Type1) 워크스페이스 : 협업 가능
	- GPU(Type2) 워크스페이스 : 협업 불가 

워크로드 : 사용자가 생성한 각 GPU 개발 환경을 의미 합니다. (가상머신, 인스턴스와 유사)
- 워크로드는 종료시 모든 파일은 삭제됩니다. (마이디스크에 저장하세요.)

리포지토리 : 개발을 위해 필요한 소스코드, 파일, 모델 들의 저장소를 의미 합니다. 
- 한번 설정한 **리포지토리**는 워크로드 생성시 호출하여 활용 가능합니다. 
- 리포지토리는 외부 저장소 연결을 지원 합니다. 

마이디스크 : 자동차데이터플랫폼에서 사용자에게 제공하는 저장공간입니다. 
- 워크로드 생성시 자동으로 연결이 되며, 이곳에 저장된 파일은 워크로드 종료시도 유지 됩니다. 

마켓 플레이스 : 자동차데이터플랫폼에서 제공하는 SaaS/APIaaS로 인공지능 개발 플랫폼에서 개발된 결과물은 마켓 플레이스를 활용하여 제공 할 수 있습니다. 

### 주요 기능과 특징



인공지능 개발 플랫폼이 제공하는 주요 기능과 특징은 다음과 같습니다.



인공지능 개발 플랫폼은 다음과 같은 특징이 있습니다.



- **인공지능 개발 플랫폼에서 제공하는 포인트를 사용해 여러 GPU 자원 사용 가능**
  - 자동차데이터플랫폼(KADaP)이 자체 보유한 GPU 자원을 제공합니다. (이하, 보유 GPU)
  - 협약기관 및 외부 기관의 GPU 자원을 DePIN 기술을 활용하여 제공합니다. (이하, 공유 GPU)

- **관리자가 설정한 GPU 스케줄링 정책에 따라 주간에는 개발 모드(IDE), 야간에는 학습 모드(Batch)로 자동 전환되어 GPU 자원을 효율적으로 사용 가능**
  - Interactive Job: AI 모델 개발 및 코드 검증을 위한 실행 환경을 제공합니다.
  - Batch Job: 대규모 데이터셋 기반의 모델 분산 학습을 위한 일괄 처리 실행 환경을 제공합니다.

- **자동차데이터플랫폼(KADaP)이 제공하는 Object Detection ML 모델을 사용해 AI 모델 학습 가능**

- **자동차데이터플랫폼(KADaP)에서 제공하는 개발 툴, 리포지토리 및 스토리지 옵션을 선택해 자신만의 개발 환경 생성 가능**
