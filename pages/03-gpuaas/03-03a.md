
# 워크로드 바로 생성하기

워크로드 생성은 **바로 생성** & **상세 생성** 두 방법이 있습니다.

**바로 생성**은 GPU 슬롯을 클릭하여 **기본 설정 값**을 활용하여 워크로드를 바로 생성 할 수 있습니다. 



[[TIP("참고")]]
사용자가 원하는 상세 설정을 적용해 워크로드를 생성하려면 [워크로드 생성하기\(상세\)](https://wikidocs.net/375269)를 참고하세요.
[[/TIP]]


워크로드를 바로 생성하려면 다음 순서대로 진행하세요.



1. 인공지능 개발 플랫폼 홈 화면에서 사용하지 않는 GPU 슬롯을 클릭하세요.

2. 왼쪽의 GPU 사용 정보에서 **워크로드 생성**을 클릭하세요.

3. 워크로드 생성창이 나타나면 필수 정보를 입력하거나 설정하세요.



![2818.png](./../../assets/03-gpuaas/2818.png)


[[TIP("참고")]]
**워크로드 상세 설정**을 클릭하면 워크로드 생성 화면으로 이동합니다. 자세한 설명은 [워크로드 생성하기\(상세\)](https://wikidocs.net/375269)를 참고하세요.
[[/TIP]]

4. **워크로드 생성**을 클릭하세요.

워크로드가 생성되면 **워크스페이스** > **워크로드** 목록 페이지로 이동해 상세 정보를 확인할 수 있습니다.
