# 리포지토리 추가


[[TIP("참고")]]
리포지토리는 워크로드 생성 단계에 필요한 Container Image, 소스코드, 데이터셋 경로, 모델 경로 정보, 스토리지를 설정하여 생성시 할 수 있습니다. 
[[/TIP]]



# 리포지토리 -  스토리지


### 스토리지 추가 및 관리하기



인공지능 개발 플랫폼에서 사용할 스토리지를 추가해 모델 개발 및 학습 시 필요한 데이터를 저장하고 관리할 수 있습니다. 사용자가 사용하는 NFS, S3 등의 외부 저장소나 자동차데이터플랫폼(KADaP)에서 제공하는 Object Storage를 추가할 수 있습니다.



#### 스토리지 목록 화면 구성



사용자가 추가한 스토리지 목록을 확인할 수 있습니다. 스토리지 목록 화면은 다음과 같이 구성됩니다.



![4268.png](./../../assets/03-gpuaas/4268.png)



| 번호 | 항목 | 설명 |

| --- | --- | --- |

| 1 | + 스토리지 추가 | 데이터를 저장할 스토리지 경로를 추가합니다. |

| 2 | 스토리지 목록 | 스토리지 목록에는 스토리지 이름과 생성 정보가 표시됩니다.<ul><li>![icon4fmt.png](./../../assets/03-gpuaas/icon4fmt.png)를 클릭하면 선택한 스토리지를 삭제할 수 있습니다.</li> <li>스토리지 상세 정보창에서는 스토리지 이름, IP 주소나 서버주소가 표시되며, 스토리지의 이름을 수정할 수 있습니다.</li></ul> |



#### 스토리지 추가



스토리지 경로를 추가하려면 다음 순서대로 진행하세요.



1. 인공지능 개발 플랫폼 홈 화면에서 메인 메뉴의 **리포지토리**를 클릭하세요.

2. 상단의 서브 메뉴에서 **스토리지**를 클릭하세요.

3. 스토리지 목록 페이지에서 **+ 스토리지 추가**를 클릭하세요.

4. 스토리지 추가창이 나타나면 상세 항목을 설정하고 **추가**를 클릭하세요.



![4303.png](./../../assets/03-gpuaas/4303.png)



| 번호 | 항목 | 설명 |

| --- | --- | --- |

| 1 | 타입 | 스토리지 타입을 선택합니다.<ul><li>**NFS**: 기존에 사용 중인 NAS 저장장치가 있는 경우 선택합니다.</li><li>**DELL Unity**: 대규모 데이터 처리를 위해 사용하는 기업 전용 DELL 스토리지가 구축되어 있는 경우 선택합니다.<ul><li>Unity를 연결하려면 스토리지 장비에서 사전 설정을 진행해야 합니다. 사전 설정에 대해서는 인공지능 개발 플랫폼 관리자에게 문의하세요.</li></ul></li><li>**Object Storage**: 대용량 데이터를 저장하거나 외부 클라우드 저장소와 연동하는 경우 선택합니다.</li><li>**협약기관 S3 Storage**: Gcube 클러스터의 S3 저장소와 연동하는 경우 선택합니다.</li></ul> |

| 2 | 스토리지 이름 | 사용할 스토리지 이름을 입력합니다. |

| 3 | IP 주소 | 스토리지 서버의 IP 주소를 입력합니다. |

| 4 | 스토리지 저장 Path | 스토리지 서버 내부에 데이터를 저장할 폴더 경로를 입력합니다. |

| 5 | 서버 주소 | Object Storage에 접속할 수 있는 API 엔드포인트 주소를 입력합니다. |

| 6 | 액세스 키 | Object Storage 접속 시 ID 역할을 하는 액세스 키를 입력합니다. |

| 7 | 시크릿 키 | Object Storage 접속 시 비밀번호 역할을 하는 시크릿 키를 입력합니다. |



#### 스토리지 활용



스토리지 정보를 등록해 놓으면 데이터셋 경로나 모델 경로를 추가할 때 스토리지를 적용할 수 있습니다.



- **리포지토리** > **데이터셋 경로** > **데이터셋 경로 추가** > **내부기관 Storage** > 등록한 스토리지 선택해 저장

&#x20; - 데이터셋 경로 추가에 대한 자세한 설명은 [데이터셋 경로 추가](#데이터셋-경로-추가)를 참고하세요.

- **리포지토리** > **모델 경로** > **모델 경로 추가** > **내부기관 Storage** > 등록한 스토리지 선택해 저장

&#x20; - 모델 경로 추가에 대한 자세한 설명은 [모델 경로 추가](#모델-경로-추가)를 참고하세요.
