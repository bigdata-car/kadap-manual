## 허브 활용하기

인공지능 개발 플랫폼에서 지원하는 ML 모델로 Batch Job 워크로드를 실행해 분산 학습을 수행할 수 있습니다

### ML 모델을 사용해 워크로드 생성하기

> ![4513.png](KADaP_UserManual_GPUaaS-web-resources/image/4513.png) **참고**
>
> 워크로드 각 항목에 대한 자세한 설명은 [워크로드 생성하기](#워크로드-생성하기)를 참고하세요.

ML 모델 이미지로 워크로드를 생성하려면 다음 순서대로 진행하세요.

1. 인공지능 개발 플랫폼 홈 화면에서 메인 메뉴의 **허브**를 클릭하세요.
2. 허브 목록에서 원하는 모델을 클릭하세요.
3. 모델 상세 페이지에서 필요한 데이터와 상세 정보를 확인하고 **+ 워크로드 생성**을 클릭하세요.

   ![4360.png](KADaP_UserManual_GPUaaS-web-resources/image/4360.png)

4. 워크스페이스 선택창에서 사용할 워크스페이스를 선택하고 **선택**을 클릭하세요.
5. 워크로드 생성 페이지에서 상세 항목을 설정하고 **워크로드 생성**을 클릭하세요.
   - [*]가 표시된 항목은 필수 항목이므로 반드시 입력하세요.

   ![4374.png](KADaP_UserManual_GPUaaS-web-resources/image/4374.png)

> ![4513.png](KADaP_UserManual_GPUaaS-web-resources/image/4513.png) **참고**
>
> - Hub 선택 시 소스코드가 지정되어 있으므로 변경하거나 삭제할 수 없습니다.
> - 데이터셋 경로와 모델 경로를 각각 1개씩만 추가할 수 있으며, 각 경로를 삭제하면 해당 경로가 지정된 필드도 함께 삭제됩니다.
> - 파라미터는 추가하거나 삭제할 수 없으며 적용된 항목의 값을 수정만 할 수 있습니다.
> - Hub 선택 시 실행 경로와 실행 명령어가 자동으로 입력되며 수정할 수 없습니다.
> - 시간 예측 파라미터 항목은 자동으로 적용되며 적용된 항목의 값을 수정만 할 수 있습니다. 입력된 파라미터 값에 따라 워크로드의 예상 종료 시간을 확인할 수 있습니다.
