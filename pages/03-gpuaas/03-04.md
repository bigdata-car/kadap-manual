\#### 공유 GPU



공유 GPU 탭에는 외부에서 공유하는 GPU 목록이 표시됩니다. GPU 슬롯을 클릭하면 다음과 같이 상세 정보가 표시됩니다.



\- \*\*공유 GPU (Type1)\*\*: 사용자가 등록한 GPU 자원 목록을 표시합니다. \*\*노드 관리\*\* 목록에 표시되는 GPU를 사용할 수 있습니다.

\- \*\*공유 GPU (Type2)\*\*: 자동차데이터플랫폼(KADaP) 협약기관에서 제공하는 GPU 자원 목록을 표시합니다. 작업에 맞는 GPU 사양을 선택해 사용할 수 있습니다.
