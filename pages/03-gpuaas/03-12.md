## 리포지토리 추가 및 관리하기 {#리포지토리-추가-및-관리하기}



Container Image를 추가하고 워크로드 생성 시 사용하는 소스코드, 데이터셋 경로, 모델 경로 정보를 추가할 수 있습니다. 또한 데이터를 저장할 수 있는 스토리지를 추가해 데이터를 저장하고 관리할 수 있습니다.



### 소스코드 경로 추가 및 관리하기



소스코드는 일반적으로 사용자가 사전에 생성한 외부 저장소에서 저장해 관리합니다. 워크로드가 실행되면 외부 저장소에 저장된 소스코드를 연결해 개발이나 학습을 진행합니다.



#### 소스코드 목록 화면 구성



사용자가 추가한 소스코드 목록을 확인할 수 있습니다. 소스코드 목록 화면은 다음과 같이 구성됩니다.



![3861.png](KADaPUserManualGPUaaS-web-resources/image/3861.png)



| 번호 | 항목 | 설명 |

| --- | --- | --- |

| 1 | 소스코드 목록 필터 | 목록 필터를 선택해 소스코드 목록에 적용합니다.<ul><li>저장소 타입: Git hub, Git lab, Docker 종류별로 표시</li></ul> |

| 2 | 검색창 | 소스코드 이름을 입력해 검색합니다. |

| 3 | + 소스코드 추가 | 외부 저장소의 소스코드 경로를 추가합니다. |

| 4 | 소스코드 목록 | 소스코드 목록에는 공개 범위 및 저장소 타입, url 경로, 기본 마운트 경로, 생성 정보가 표시됩니다. <ul><li>소스코드 상세 정보 페이지에서는 소스코드 경로 기본 정보와 크레덴셜 사용 여부, 실행 명령어와 파라미터 설정 항목이 표시됩니다. 또한 소스코드의 정보를 수정하거나 삭제할 수 있습니다.</li></ul>|



#### 소스코드 경로 추가 {#소스코드-경로-추가}



사용자가 사전에 생성한 소스코드를 워크로드에 연결할 수 있습니다.



사용자의 소스코드 경로를 추가하려면 다음 순서대로 진행하세요.



1. 인공지능 개발 플랫폼 홈 화면에서 메인 메뉴의 **리포지토리**를 클릭하세요.

2. 상단의 서브 메뉴에서 **소스코드**를 클릭하세요.

3. 소스코드 목록 페이지에서 **+ 소스코드 추가**를 클릭하세요.

4. 소스코드 추가창이 나타나면 상세 항목을 입력하고 **추가**를 클릭하세요.

 - 소스코드가 추가가 완료되면 소스코드 목록에 표시됩니다.



![3892.png](KADaPUserManualGPUaaS-web-resources/image/3892.png)



> ![4513.png](KADaPUserManualGPUaaS-web-resources/image/4513.png)**참고**

>

> - 저장소 타입은 Git hub, Git lab, Bitbucket 중에서 선택할 수 있습니다.

> - 외부 저장소의 Private repository 경로를 등록할 경우에는 크레덴셜 항목을 설정해야 합니다. **설정** > **크레덴셜**에 등록한 크레덴셜을 선택해 적용하세요.

>   - 크레덴셜을 새로 등록해야 한다면 **+ Credential 등록**을 클릭합니다. 크레덴셜 추가에 대한 자세한 설명은 [크레덴셜 추가하기](#크레덴셜-추가하기)를 참고하세요.



#### 소스코드 활용



소스코드 경로를 추가하면 **워크스페이스** > **공유 리포지토리**에 추가해 워크스페이스에 등록한 멤버와 공유할 수 있으며 워크로드 생성 시 적용할 수 있습니다.



**공유 리포지토리에 추가해 멤버와 공유**



워크스페이스에 추가된 멤버들과 소스코드를 공유할 수 있습니다.



공유 리포지토리에 소스코드를 추가하려면 다음 순서대로 진행하세요.



1. 인공지능 개발 플랫폼 홈 화면에서 메인 메뉴의 **워크스페이스**를 클릭하세요.

2. 워크스페이스 목록 페이지가 나타나면 소스코드를 추가할 워크스페이스를 클릭하세요.

3. 상단의 서브 메뉴에서 **워크스페이스 설정**을 클릭하세요.

4. 워크스페이스 정보 페이지에서 왼쪽의 **공유 리포지토리**를 클릭하세요.

5. 공유 리포지토리 페이지에서 **+ 공유 소스코드 추가**를 클릭하세요.

6. 공유 소스코드 추가창이 나타나면 등록한 소스코드를 선택하고 기본 마운트 경로를 입력한 후 **확인**을 클릭하세요.



**워크로드 생성 시 적용**



사용자가 저장한 소스코드를 활용해 워크로드를 생성하려면 다음 순서대로 진행하세요.



1. 인공지능 개발 플랫폼 홈 화면에서 메인 메뉴의 **워크스페이스**를 클릭하세요.

2. 워크스페이스 목록 페이지가 나타나면 워크로드를 생성할 워크스페이스를 클릭하세요.

3. 상단의 서브 메뉴에서 **워크로드**를 클릭하세요.

4. 워크로드 목록 페이지에서 **+ 워크로드 생성**을 클릭하세요.

5. Task에서 **+ 소스코드 추가**를 클릭하세요.

6. 소스코드 추가창이 나타나면 **타입** > **워크스페이스**나 **사용자**를 선택하세요.

 - **직접 입력**을 선택해 추가하는 소스코드는 해당 워크로드에서만 사용되는 일회용 소스코드입니다. 소스코드를 저장해 사용하려면 [소스코드 경로 추가](#소스코드-경로-추가)를 참고하세요.

7. 소스코드 목록에서 적용할 소스코드를 선택하고 상세 항목 설정 후 **확인**을 클릭하세요.



![3930.png](KADaPUserManualGPUaaS-web-resources/image/3930.png)



8. 워크로드 생성 페이지에서 상세 항목을 설정하고 **워크로드 생성**을 클릭하세요.

- 사용자가 등록한 경로의 소스코드가 워크로드에 연결되어 실행됩니다.
