\## 리포지토리 추가 및 관리하기 {#리포지토리-추가-및-관리하기}



Container Image를 추가하고 워크로드 생성 시 사용하는 소스코드, 데이터셋 경로, 모델 경로 정보를 추가할 수 있습니다. 또한 데이터를 저장할 수 있는 스토리지를 추가해 데이터를 저장하고 관리할 수 있습니다.



\### 모델 경로 추가 및 관리하기



사용자가 생성한 AI 모델의 가중치 파일이나 설정 파일, 체크포인트 파일 등을 자동차데이터플랫폼(KADaP)의 파일 스토리지에 직접 업로드하거나 외부의 저장 공간에 데이터를 보관하는 경우 등록한 경로의 데이터를 워크로드에서 실행할 수 있습니다.



\#### 모델 경로 목록 화면 구성



사용자가 추가한 모델 경로 목록을 확인할 수 있습니다. 모델 경로 목록 화면은 다음과 같이 구성됩니다.



!\[4152.png](KADaP\_UserManual\_GPUaaS-web-resources/image/4152.png)



| 번호 | 항목 | 설명 |

| --- | --- | --- |

| 1 | 모델 경로 목록 필터 | 목록 필터를 선택해 모델 경로 목록에 적용합니다.<ul><li>생성일자 순, 이름 순으로 표시</li> <li>Storage type 종류별로 표시</li></ul> |

| 2 | 검색창 | 모델 경로 이름을 입력해 검색합니다. |

| 3 | + 모델 경로 추가 | 데이터가 저장된 모델 경로를 추가합니다. |

| 4 | 모델 경로 목록 | 모델 경로 목록에는 Storage type과 모델 경로 이름, 생성 정보가 표시됩니다.<ul><li>모델 경로 상세 정보 페이지에서는 기본 정보, 설정 내용, 생성 정보와 워크로드 정보가 표시됩니다. 또한 모델 경로의 정보를 수정하거나 삭제할 수 있습니다.</li></ul>|



\#### 모델 경로 추가 {#모델-경로-추가}



저장소의 모델 경로를 등록해 워크로드 생성 시 적용할 수 있습니다.



모델 경로를 추가하려면 다음 순서대로 진행하세요.



1\. 인공지능 개발 플랫폼 홈 화면에서 메인 메뉴의 \*\*리포지토리\*\*를 클릭하세요.

2\. 상단의 서브 메뉴에서 \*\*모델 경로\*\*를 클릭하세요.

3\. 모델 경로 목록 페이지에서 \*\*+ 모델 경로 추가\*\*를 클릭하세요.

4\. 데이터가 저장되어 있는 저장소 타입을 선택하세요.



&#x20;  !\[4179.png](KADaP\_UserManual\_GPUaaS-web-resources/image/4179.png)



5\. 선택한 저장소에 따라 상세 항목을 입력하고 \*\*추가\*\*를 클릭하세요.



&#x20;  !\[4191.png](KADaP\_UserManual\_GPUaaS-web-resources/image/4191.png)



| 번호 | 항목 | 설명 |

| --- | --- | --- |

| 1 | 스토리지 목록 | 모델 데이터를 업로드하거나 연결할 스토리지 목록을 선택합니다.<ul><li>내부기관을 선택한 경우 모델 데이터를 업로드할 자동차데이터플랫폼(KADaP) 내부 스토리지를 선택합니다.</li><li>S3 스토리지를 선택한 경우 등록한 버킷 폴더 중 사용할 항목을 선택합니다.</li></ul> |

| 2 | 모델 경로 이름 | 사용할 모델 경로 이름을 입력합니다. |

| 3 | Default mount path | 워크로드가 실행될 때 연결할 컨테이너 내부 경로를 입력합니다. |

| 4 | 파일 업로드 | 사용자 PC에 저장되어 있는 모델 데이터 파일을 드래그하거나 \*\*파일 선택\*\*을 클릭해 업로드합니다.<ul><li>최대 2 GB 크기의 모델 데이터 파일을 업로드할 수 있습니다.</li></ul> |

| 5 | Server IP | 데이터가 저장된 외부 NFS 스토리지 서버의 IP 주소를 입력합니다. |

| 6 | Server Path | 데이터가 저장된 외부 NFS 스토리지 서버의 디렉토리 경로를 입력합니다. |



> !\[4513.png](KADaP\_UserManual\_GPUaaS-web-resources/image/4513.png) \*\*참고\*\*

>

> - 모델 경로를 등록하면 \*\*워크스페이스\*\* > \*\*공유 리포지토리\*\*에 추가해 워크스페이스에 등록한 멤버와 공유할 수 있으며, 워크로드 생성 시 적용할 수 있습니다.

> - \*\*내부기관 스토리지\*\*를 선택한 경우, 왼쪽의 \*\*파일 목록\*\* 메뉴를 사용할 수 있습니다. 선택한 경로에 저장된 파일을 확인하고, 파일을 업로드하거나 다운로드할 수 있습니다. 파일 목록에 대한 자세한 설명은 \[파일 목록](#파일-목록)을 참고하세요.



\#### 모델 활용



모델 경로를 생성하면 \*\*워크스페이스\*\* > \*\*공유 리포지토리\*\*에 추가해 워크스페이스에 등록한 멤버와 공유할 수 있으며 워크로드 생성 시 적용할 수 있습니다.



\##### 공유 리포지토리에 추가해 멤버와 공유



워크스페이스에 추가된 멤버들과 공유할 모델 데이터를 공유할 수 있습니다.



공유 리포지토리에 소스코드를 추가하려면 다음 순서대로 진행하세요.



1\. 인공지능 개발 플랫폼 홈 화면에서 메인 메뉴의 \*\*워크스페이스\*\*를 클릭하세요.

2\. 워크스페이스 목록 페이지가 나타나면 소스코드를 추가할 워크스페이스를 클릭하세요.

3\. 상단의 서브 메뉴에서 \*\*워크스페이스 설정\*\*을 클릭하세요.

4\. 워크스페이스 정보 페이지에서 왼쪽의 \*\*공유 리포지토리\*\*를 클릭하세요.

5\. 공유 리포지토리 페이지에서 \*\*+ 공유 모델 경로 추가\*\*를 클릭하세요.

6\. 공유 모델 경로 추가창이 나타나면 등록한 모델 경로를 선택하고 기본 마운트 경로를 입력한 후 \*\*추가\*\*를 클릭하세요.



\##### 워크로드 생성 시 적용



사용자가 저장한 모델 경로를 적용해 워크로드를 생성하려면 다음 순서대로 진행하세요.



1\. 인공지능 개발 플랫폼 홈 화면에서 메인 메뉴의 \*\*워크스페이스\*\*를 클릭하세요.

2\. 워크스페이스 목록 페이지가 나타나면 워크로드를 생성할 워크스페이스를 클릭하세요.

3\. 상단의 서브 메뉴에서 \*\*워크로드\*\*를 클릭하세요.

4\. 워크로드 목록 페이지에서 \*\*+ 워크로드 생성\*\*을 클릭하세요.

5\. Task에서 \*\*+ 모델 경로 추가\*\*를 클릭하세요.

6\. 모델 경로 추가창이 나타나면 \*\*타입\*\* > \*\*워크스페이스\*\*나 \*\*사용자\*\*를 선택하세요.

7\. 모델 경로 목록에서 공유 모델 경로를 선택하고 마운트 경로를 입력한 후 \*\*확인\*\*을 클릭하세요.



&#x20;  !\[4253.png](KADaP\_UserManual\_GPUaaS-web-resources/image/4253.png)



8\. 워크로드 생성 페이지에서 상세 항목을 설정하고 \*\*워크로드 생성\*\*을 클릭하세요.

&#x20;  - 사용자가 등록한 모델 데이터가 워크로드에 연결되어 실행됩니다.



\### 스토리지 추가 및 관리하기



인공지능 개발 플랫폼에서 사용할 스토리지를 추가해 모델 개발 및 학습 시 필요한 데이터를 저장하고 관리할 수 있습니다. 사용자가 사용하는 NFS, S3 등의 외부 저장소나 자동차데이터플랫폼(KADaP)에서 제공하는 Object Storage를 추가할 수 있습니다.
