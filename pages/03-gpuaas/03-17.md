## 노드 관리

사용자가 추가한 노드를 확인하고 관리할 수 있습니다. 사용자가 보유한 로컬 서버나 GPU 자원을 노드로 등록하면 AI 모델 개발 및 학습에 활용할 수 있습니다.

### 내 노드 관리하기

내 노드 목록에서 공개 범위 변경, GPU 분할 등을 설정할 수 있습니다.

#### 공개 범위 설정

**내 노드 목록** > **공개 범위** 항목을 클릭해 노드의 공개 범위를 변경할 수 있습니다.

- 노드 공개 범위 설정창이 나타나면 원하는 공개 범위를 선택하고 **변경**을 클릭합니다.

  ![4523.png](KADaP_UserManual_GPUaaS-web-resources/image/4523.png)

#### GPU 분할 설정

**내 노드 목록** > **GPU 분할** 항목을 클릭해 노드의 GPU 분할 설정을 할 수 있습니다.

- GPU 분할 설정창이 나타나면 분할 개수를 선택하고 **설정**을 클릭합니다.
  - **미지원**: GPU 분할을 지원하지 않는 노드는 미지원으로 표시됩니다. 버튼이 비활성화되어 선택할 수 없습니다.
  - **MPS** (Multi-Process Service): GPU를 분할하지 않고 하나의 GPU에서 사용되지 않는 자원을 여러 연산 작업 시 나눠서 사용하는 방식으로 소규모 AI 모델 추론 작업 여러 개를 동시에 실행할 수 있습니다.
  - **MIG** (Multi-Instance GPU): 하나의 GPU를 독립된 가상 GPU 인스턴스로 분할하는 방식으로 고성능 GPU에 적용해 대규모 모델 학습을 실행할 수 있습니다.

  ![4539.png](KADaP_UserManual_GPUaaS-web-resources/image/4539.png)

#### 노드 개인 정보 등록 설정

**내 노드 목록** > **정보 추가** > **추가**를 클릭해 사용자 정보를 추가하거나 등록 해제할 수 있습니다.

- 추가: 노드 개인 정보 등록창이 나타나면 개인 정보를 확인하고 **등록**을 클릭합니다.
- 등록 해제: 노드 개인 정보 등록창이 나타나면 **등록 해제**를 클릭합니다.
