# 마켓 플레이스 소개



마켓 플레이스는 KADaP 내에서 운영되는 데이터 기반 거래 공간으로, 기업과 기관이 데이터를 공유하거나 판매하고, 필요한 데이터를 구매하는 등 데이터 상품의 거래와 서비스 환경을 제공합니다.

## 주요 기능과 특징



### API 마켓 플레이스



- **API 신청 · 구독 · 호출 기능 제공**



&#x20; 사용자가 원하는 API를 선택해 신청하고, 구독 후 바로 호출할 수 있습니다.



- **표준화된 서비스 구조**



&#x20; API 문서와 요청/응답 규격, 인증 체계를 통일된 방식으로 구성해 사용자 편의성을 높였습니다.



- **승인 기반 보안 체계**



&#x20; 개발자 및 관리자의 승인 절차를 거쳐 안전하게 API를 이용할 수 있는 환경을 제공합니다.



- **사용 통계 및 구독 관리 대시보드**



&#x20; API 호출 현황, 응답 상태, 에러 비율, 사용 신청 내역 등 API 사용 현황을 직관적으로 파악할 수 있습니다.
