\### API Key 관리하기



API 마켓 플레이스에서는 승인된 모든 API 테스트에 사용할 수 있는 공용 API Key를 제공합니다. 공용 API Key는 여러 API에서 공통으로 사용되기 때문에 노출이나 오용되지 않도록 주의하시기 바랍니다.



\#### 재발급



API Key가 노출되었거나 보안 상의 이유로 새 Key가 필요한 경우, \*\*재발급\*\* 기능을 이용해 새로운 API Key를 생성할 수 있습니다. 재발급 후 기존 Key는 즉시 사용 중단되며, 새로 발급된 Key로만 API 호출이 가능합니다.



!\[재발급 화면](./../../assets/04-market-app/9353.png)



\#### 구독취소



API Key는 사용자가 구독취소 전까지 지속적으로 사용할 수 있습니다. API가 불필요한 경우 \*\*구독취소\*\*를 통해 해당 API 사용을 해지할 수 있습니다. 구독취소 후에는 API Key가 비활성화되며, API 호출이 불가능합니다.



!\[구독취소 화면](./../../assets/04-market-app/9385.png)



\#### 보안 주의사항



API Key는 외부에 노출될 경우 부정 사용 또는 오용 위험이 있으므로 다음 보안 사항을 준수해야 합니다.



\- API Key는 외부에 공유하거나 저장소(깃허브 등)에 공개하지 않습니다.

\- 노출이 의심되는 경우 즉시 재발급하여 기존 Key를 폐기합니다.

\- 클라이언트 애플리케이션 등에서 직접 노출되지 않도록 서버 환경에서 안전하게 관리합니다.

\- 불필요한 Key는 구독 취소하여 보안 위험을 최소화합니다.
