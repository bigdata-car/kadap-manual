\## API 게시 요청하기



API 배포가 완료되면 마켓 플레이스에서 사용할 수 있도록 게시 요청을 진행할 수 있습니다.



API 상세 페이지 상단에서 \*\*게시요청\*\*을 클릭하면, API 목록에서 해당 API의 상태가 \*\*게시대기\*\*로 변경됩니다.



!\[게시 요청 화면](./../../assets/04-market-app/10624.png)



\### API 게시 요청 조건 및 절차



API 게시 요청은 다음 조건을 만족해야 합니다.



\- API는 메서드와 버전이 생성되어야 게시 요청이 가능합니다.

\- 게시 요청 후에는 관리자 검토 및 승인이 필요합니다.

\- 관리자가 승인을 완료하면 API가 마켓 플레이스에 공개되어 실제 사용자가 호출할 수 있습니다.



\### API 게시 요청 상태 확인하기



게시 요청 후에는 \*\*API 관리\*\* > API 목록에서 게시 상태를 확인할 수 있습니다.



| 게시 상태 | 설명 |

|-----------|------|

| 게시대기 | 게시 요청이 완료되었고, 관리자의 승인 대기 중에 있습니다. |

| 게시완료 | 관리자가 승인을 완료했고, API가 마켓 플레이스에 공개되었습니다. |

| 게시거부 | 게시가 거부된 상태입니다. 게시 거부 사유는 메일로 발송됩니다.<br>문제를 해결한 후, 다시 게시 요청을 진행할 수 있습니다. |



> !\[8991.png](./../../assets/04-market-app/8991.png) \*\*참고\*\*

>

> 게시가 완료된 API의 게시를 취소하려면 API 상세 페이지 상단의 \*\*게시취소\*\*를 클릭하세요. 이때 해당 API를 구독 중인 사용자가 없는 경우에만 게시 취소가 가능합니다.
