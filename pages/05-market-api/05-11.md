\### 버전 등록하기



API 버전은 API의 변경 사항을 관리하기 위해 필요한 단계입니다. API 버전을 등록하면 엔드포인트 설정이 엔진에 배포되며, 배포가 완료되면 최종 서비스 URL이 생성됩니다.



1\. \*\*버전\*\* 탭에서 \*\*버전 생성\*\*을 클릭하세요.



&#x20;  !\[버전 생성 화면](./../../assets/04-market-app/10340.png)



2\. 버전 정보를 입력하세요.

&#x20;  - \*\*이름\*\*: 등록할 버전 이름을 입력합니다. (예: V01, Ver1.0 등)

&#x20;  - \*\*IP 제한\*\*: API 호출을 허용할 IP 주소 범위를 지정하여 특정 IP에서만 접근할 수 있도록 설정합니다.



&#x20;  !\[버전 정보 입력 화면](./../../assets/04-market-app/10374.png)



3\. 모든 정보를 입력한 후, \*\*저장\*\*을 클릭하세요.



버전 생성이 완료되었습니다. 최초 버전인 경우에는 생성과 동시에 자동으로 배포까지 진행됩니다.



\## API 수정/삭제/복사하기



API 상세 페이지 상단의 !\[설정 아이콘콘](./../../assets/04-market-app/10439.png) 옵션을 통해 생성된 API를 수정하거나 삭제할 수 있으며, 기존 API를 복사하여 새로운 API 생성에 활용할 수도 있습니다.



!\[API 수정/삭제/복사 옵션 화면](./../../assets/04-market-app/10424.png)



\### API 수정 및 삭제하기



API 이름, 설명, 버전 정보, 메서드 정의 등 기존에 설정한 항목을 변경할 수 있습니다.<br>설정을 변경한 후, \*\*저장\*\*을 클릭하면 해당 API에 변경된 내용이 바로 적용됩니다.



사용하지 않은 API는 삭제할 수 있습니다. API 삭제 시 관련 메서드 및 구독 정보도 함께 삭제되므로 삭제하기 전에 반드시 해당 API의 구독 상태를 확인하세요.



\### API 복사하기



기존 API 설정을 기반으로 새로운 API를 간편하게 생성할 수 있으며, 유사한 구조의 API를 빠르게 만들 때 유용합니다.
