## (개발자) API 등록하기



API 등록 기능은 개발자가 직접 제공하는 API를 API 마켓 플레이스에 등록하여 사용자에게 공개할 수 있도록 하는 기능입니다. 등록된 API는 승인 과정을 거쳐 마켓 플레이스에 노출되며, 사용자는 해당 API에 대해 사용 신청을 진행할 수 있습니다. API 제공자는 등록된 API의 정보, 버전, 사용 현황 등을 관리할 수 있습니다.



> ![8991.png](./../../assets/04-market-app/8991.png) **참고**

>

> API 등록과 관리는 **개발자 센터**에서 수행하며, **개발자 센터**를 이용하려면 개발자 권한이 필요합니다.<br>개발자 권한 신청은 [개발자 등록](#api-개발자-등록)를 참고하세요.



### API 등록 절차



API를 등록하려면 다음 순서대로 진행하세요.



![API 등록 절차 흐름도](./../../assets/04-market-app/9404.png)



- **타겟 서버 그룹 및 타겟 서버 등록**



API가 연결될 서버 그룹을 생성하고, 서버를 추가한 후 로드밸런싱 방식을 설정합니다.



- **API 등록**



API 이름, 설명, 인증 방식 등 기본 정보를 입력해 API를 생성합니다.



- **메서드 등록** 



&#x20; API 요청을 처리할 HTTP 메서드와 경로를 지정하고 요청 및 응답 스펙을 정의합니다.



- **버전 등록**



&#x20; API 변경 사항을 관리하기 위해 새로운 버전을 생성합니다. 필요 시 다중 버전을 운영할 수 있습니다.



- **API 배포**



&#x20; 설정이 완료된 API를 선택한 환경에 반영하여 실제 서비스에 배포합니다.



- **API 게시 요청**



&#x20; 마켓 플레이스에 API를 노출하기 위한 게시를 요청합니다. 게시된 API는 관리자 승인 후 마켓 플레이스에 공개됩니다.
