\### API 호출 테스트하기



API 사용 승인이 완료되면 API 호출 테스트를 진행할 수 있습니다.



1\. \*\*마이페이지\*\* > \*\*API 신청 현황\*\*에서 해당 API를 클릭하세요.



2\. 상세 화면에서 \*\*API DOCS 바로가기\*\*를 클릭하세요.



&#x20;  !\[API DOCS 바로가기 화면](./../../assets/04-market-app/9267.png)



3\. Parameters와 Request body 영역에 필요한 값을 입력하세요.

&#x20;  - 필수 값(\*)이 표시된 항목은 반드시 입력해야 합니다.

&#x20;  - 선택 값은 필요에 따라 입력합니다.



&#x20;  !\[파라미터 입력 화면](./../../assets/04-market-app/9329.png)



4\. 해당 값을 모두 입력 후, \*\*Execute\*\*를 클릭하세요.

&#x20;  - 요청과 응답 결과가 화면에 표시되며, 응답 상태 코드와 반환 데이터를 확인할 수 있습니다.



> !\[8991.png](./../../assets/04-market-app/8991.png) \*\*참고\*\*

>

> - 사용 신청 전이라도 API가 정상적으로 작동하는지 최대 10회까지 테스트할 수 있습니다.

>   - API 검색 > 호출 테스트할 API 클릭 > \*\*API DOCS 바로가기\*\* > \*\*Test Key 발급\*\* > 호출 테스트 시작

> - API 문서는 버전별로 json 포맷으로 제공합니다.
