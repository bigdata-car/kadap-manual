\# API 마켓 플레이스



API 마켓 플레이스에서는 API 사용 신청, 키 발급, 사용 이력 조회뿐만 아니라, 직접 개발한 API를 등록하고 관리할 수 있는 다양한 기능을 제공합니다.



\## 메뉴 및 화면 구성



API 마켓 플레이스의 메인 화면은 다음과 같이 구성됩니다.



!\[API 마켓 플레이스 메인 화면](./../../assets/04-market-app/8700.png)



| 번호 | 항목 | 설명 |

|---|------|------|

| 1 | API | 마켓에 등록된 API가 카테고리별로 분류되어 표시됩니다. |

| 2 | 고객지원 | 마켓 이용과 관련된 공지사항, 자주 묻는 질문(FAQ), 문의사항 등을 확인할 수 있습니다. |

| 3 | 개발자 등록/개발자 센터 | API를 등록하거나 등록된 API를 관리하려면 개발자 등록이 필요합니다.<br>개발자 등록 신청이 승인되면 개발자 센터에 접속할 수 있고, 개발자 센터를 통해 API 등록과 관리에 필요한 전반적인 기능을 이용할 수 있습니다. |

| 4 | 마이페이지 | API 신청 현황 확인, 문의 작성, 내 정보 수정 등의 기능을 이용할 수 있습니다. |

| 5 | 알림 | API 사용 신청 결과, 최신 소식, 문의 사항에 대한 알림을 확인할 수 있습니다. |

| 6 | 검색 | 마켓에 등록된 API를 검색어를 통해 검색할 수 있습니다. |

| 7 | API 목록 | 마켓에 등록된 API 목록이 표시됩니다. |



\### API



마켓에 등록된 API는 카테고리별로 분류되어 있어 쉽게 찾아볼 수 있습니다. 또한 검색창에 키워드를 입력하여 원하는 API를 바로 검색할 수도 있습니다. 검색 결과는 리스트 또는 카드 형식으로 표시할 수 있으며, 최신순 또는 인기순으로 정렬할 수 있습니다.



\- \*\*전체 API\*\*: 마켓에 등록된 전체 API 목록입니다.

\- \*\*서비스 API\*\*: 입력된 데이터를 처리하고, AI 기반의 분석 및 예측 기능을 수행하는 API 목록입니다.

\- \*\*자동차산업특화\*\*: 차량 번호 기반 이력 조회, 운전면허증 확인, 차량 이미지 인식 등 자동차 서비스 현장에서 활용 가능한 실무 중심의 API 목록입니다.

\- \*\*AI/알고리즘\*\*: 다양한 대형 언어 모델(LLM)과 이미지 분석, 텍스트 음성 변환(TTS) 등 AI 기반 기능을 손쉽게 연동할 수 있는 API 목록입니다.

\### 고객지원

\#### 개발자 등록 {#api-개발자-등록}



> !\[8991.png](./../../assets/04-market-app/8991.png) \*\*참고\*\*

>

> 개발자 등록은 기업 및 기관 회원만 가능합니다.

\### 마이페이지



API 신청 현황 확인, 문의 내역, 내 정보를 확인할 수 있습니다.



\#### API 신청 현황



사용 신청한 API 목록의 상태 정보를 확인할 수 있습니다.



!\[API 신청 현황 화면](./../../assets/04-market-app/8938.png)



| 번호 | 항목 | 설명 |

|---|------|------|

| 1 | API KEY | 개발자가 승인한 API를 호출할 때 접근 권한을 인증하기 위해 사용합니다. 이 KEY는 승인된 모든 API를 테스트할 때 활용할 수 있는 공용 키입니다. |

| 2 | KEY 복사 | 화면에 표시된 API KEY 값을 사용자가 간편하게 클립보드에 복사할 수 있도록 제공되는 기능입니다. |

| 3 | 재발급 | 기존에 발급된 API KEY를 폐기하고 새로운 API KEY를 발급합니다. API KEY가 외부에 노출되는 등의 보안상 필요한 경우 API KEY를 재발급할 수 있습니다.<ul><li>재발급 후에는 기존 키로는 API를 호출할 수 없으므로, 기존 키를 사용하던 시스템에서는 새 키로 반드시 업데이트해야 합니다.</li></ul> |

| 4 | 사용 신청한 API 목록 | <ul><li>사용자가 신청한 API의 상태를 확인할 수 있고, \*\*구독취소\*\*를 통해 해당 API에 대한 사용 권한을 해지할 수 있습니다.<ul><li>구독취소 시 해당 API는 더 이상 호출할 수 없으며, 필요할 경우 다시 신청해야 합니다.</li></ul></li></ul><ul><li>각 API 목록을 클릭하면 해당 API의 상세 정보를 확인할 수 있습니다.<br>!\[사용 신청한 API 상세 화면](./../../assets/04-market-app/9119.png)</li></ul></ul> |



\#### 문의내역



API 마켓 플레이스 이용 관련 문의를 작성할 수 있으며, 등록된 문의는 목록에서 확인할 수 있습니다. 여기에서 작성한 문의는 \*\*고객지원\*\* > \*\*고객문의\*\* 메뉴에도 함께 게시됩니다.



\#### 내 정보



회원 기본 정보와 개발자 인증 정보 등을 확인하고 수정할 수 있습니다. 내 정보 수정은 \*\*자동차 데이터 포털\*\* 사이트로 이동하여 진행합니다.



\### 알림



API 사용 신청 승인 결과, 고객지원 관련 안내 등 사용자 계정과 서비스 이용에 필요한 다양한 알림을 확인할 수 있습니다. 알림 목록은 최신순으로 정렬되며, 필요 시 \*\*전체 알림 확인\*\* 기능을 사용할 수 있습니다.
