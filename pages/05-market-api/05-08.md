\### 타겟 서버 그룹 등록하기 {#타겟-서버-그룹-등록하기}



API를 연결하기 위해 먼저 타겟 서버 그룹을 생성해야 합니다. 타겟 서버 그룹은 실제 서비스 엔드포인트를 논리적으로 묶어 관리하기 위한 단위이며, 생성된 그룹은 로드밸런싱 설정과 그룹 단위 모니터링 기능을 제공합니다.



타겟 서버 그룹을 등록하려면 화면 상단의 \*\*개발자 센터\*\*에 접속한 후, 다음 순서대로 진행하세요.



1\. \*\*타겟 서버 관리\*\* 메뉴를 클릭한 후, \*\*타겟 서버 그룹 등록\*\*을 클릭하세요.



&#x20;  !\[타겟 서버 그룹 등록 메뉴 화면](./../../assets/04-market-app/9472.png)



2\. 등록할 타겟 서버 그룹의 정보를 입력하세요.

&#x20;  - \*\*그룹명/설명\*\*: 타겟 서버 그룹의 이름과 설명을 입력합니다.

&#x20;  - \*\*인증\*\*: API의 보안을 위해 필요한 인증서를 등록하거나 업데이트할 수 있습니다.

&#x20;  - \*\*로드밸런싱 알고리즘\*\*: 로드밸런싱 방식을 선택합니다.



&#x20;  !\[타겟 서버 그룹 정보 입력 화면](./../../assets/04-market-app/9517.png)



3\. \*\*저장\*\*을 클릭하세요.



타겟 서버 그룹 등록이 완료되었습니다.<br>다음 절차에 따라 등록된 타겟 서버 그룹에 타겟 서버를 등록하세요.



\### 타겟 서버 등록하기



타겟 서버 그룹에 실제로 사용할 엔드포인트(타겟 서버)를 등록할 수 있습니다.



1\. 타겟 서버 그룹 상세 화면에서 \*\*타겟 서버 등록\*\*을 클릭하세요.



&#x20;  !\[타겟 서버 등록 화면](./../../assets/04-market-app/9580.png)



2\. 등록할 타겟 서버의 세부 정보를 입력하세요.

&#x20;  - 서버에 접근하기 위한 호스트(IP 또는 도메인)와 포트 번호를 입력합니다.

&#x20;  - 연결 검증을 위해 \*\*Test Connection\*\*을 클릭하여 정상적으로 통신이 가능한지 확인합니다.



&#x20;  !\[타겟 서버 세부 정보 입력 화면](./../../assets/04-market-app/9564.png)



3\. 연결 검증이 완료되면 \*\*확인\*\*을 클릭하세요.



타겟 서버 등록이 완료되었습니다.<br>다음 절차에 따라 API를 등록하세요.
